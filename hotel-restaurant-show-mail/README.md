# ホテル・レストラン・ショー フォローメール 下書き作成ツール（GAS）

スプレッドシートの名刺データ（社名・氏名・メールアドレス）を差し込み、
**Gmail に下書き（Draft）を作成**する Google Apps Script です。
※ 送信はしません。作成後に Gmail の「下書き」で内容を確認してから送ってください。

- **CC**: `biz@temairazu.com` を全件に付与
- **件名**: `ホテル・レストラン・ショー御礼と打ち合わせのお願い`（`CONFIG.SUBJECT` で変更可）
- **会社名の差し込み**: 法人名（D列）を優先、なければ施設名（B列）
- **名前の差し込み**: 姓（I列）＋ 名（J列）
- **メールアドレス空欄の行**: スキップし、実行後のログ／ダイアログに一覧表示

## セットアップ

1. 元の Excel ファイルを Google スプレッドシートとして開く（または取り込む）。
   - 列レイアウトは元ファイルのままでOK（見出しは3行目、データは6行目から）。
2. スプレッドシート上部メニュー **拡張機能 → Apps Script** を開く。
3. `コード.gs` の中身を貼り付けて保存。
4. スプレッドシートに戻り、**再読み込み**するとメニュー **「メール下書き」** が表示される。

## 使い方

1. **メール下書き → ① プレビュー（下書きは作らない）**
   - 差し込み結果（宛先・会社名・名前）と、対象/スキップ件数を確認。
2. 問題なければ **メール下書き → ② Gmail下書きを作成**
   - 全件の下書きが作成される。Gmail の「下書き」フォルダで最終確認 → 送信。

初回実行時に Gmail / スプレッドシートへのアクセス許可（承認）を求められます。
実行アカウント自身の Gmail に下書きが作成されます。

## 設定（`CONFIG`）

| 項目 | 既定値 | 説明 |
|------|--------|------|
| `SHEET_NAME` | `''`（空=アクティブシート） | 対象シート名 |
| `HEADER_ROW` | `3` | 見出し行 |
| `DATA_START_ROW` | `6` | データ開始行（テストデータ・注釈行の次） |
| `CC` | `biz@temairazu.com` | 全件に付与するCC |
| `SUBJECT` | 上記件名 | メール件名 |

列は見出し名（`施設名`・`法人名`・`姓`・`名`・`メールアドレス`）で自動マッピングします。
列の並びが変わっても見出し名が一致していれば動作します。
